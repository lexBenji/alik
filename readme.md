# Alik Rewritten

Alik programming language but rewritten.

# Examples

Hello World in ALIK
```
writeln "Hello World"
```

Declare and call variable in ALIK
```
var int 10
writeln $i
var flt 10.5
writeln $flt
var bool true
writeln $bool
var str "hello world"
writeln $str
```

Infinite loop in ALIK that prints 'Hello World' then wait 1 second forever
```
loop
writeln "Hello World"
wait 1
end
```
